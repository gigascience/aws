The diversity of two African populations

SmileFinder is a tool for detects diversity and divergence patterns associated with selection sweeps by evaluating allele frequencies. The documentation on this page shows how the collection of SmileFinder scripts (which have been deployed as tools in GigaGalaxy) can be used within a workflow to identify a difference in diversity between two African populations.
